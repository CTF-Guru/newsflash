Jul 15, 2026Important

AsyncAPI npm packages backdoored through a CI flaw

VulnerabilitiesAudit nowDevelopers, AppSec, Platform teams

A malicious pull request abused a misconfigured pull_request_target workflow in asyncapi/generator to expose the asyncapi-bot personal access token and push trojanized releases, including @asyncapi/specs 6.11.2 and @asyncapi/generator 3.3.1, carrying a Miasma loader. The payload fires at module-load time, so installing with --ignore-scripts is no defence.

Why it matters: The release pipeline was the attack surface, not the source code - and an import-time payload defeats the --ignore-scripts habit a lot of teams rely on.

Jul 14, 2026Important

SonicWall SMA1000 zero-days exploited in the wild

VulnerabilitiesActively exploited / KEVSecurity, IT Ops

CVE-2026-15409, a critical unauthenticated SSRF, and CVE-2026-15410, a post-auth code injection, are being chained against SMA1000 appliances to extract credentials, session databases and TOTP multi-factor seeds. CISA added both to its KEV catalog and set a July 17 deadline for federal agencies; fixed firmware is v12.4.3-03453 and 12.5.0-02835.

Why it matters: Remote-access appliances remain the front door for initial access, and stolen MFA seeds keep working after a password reset.

Jul 1, 2026Potentially transformative

An AI agent ran the ransomware kill chain

AI SecurityResearch disclosedSecurity, Developers, Platform teams

Sysdig documented JADEPUFFER, in which an LLM agent chained a Langflow unauthenticated RCE (CVE-2025-3248) and a Nacos auth bypass (CVE-2021-29441) to carry out database extortion. Humans still provisioned the C2 infrastructure and chose the target; the agent handled recon, lateral movement and destruction, once turning a failed login into a correct multi-step fix in 31 seconds.

Why it matters: The finding is about the skill bar, not the novelty: Sysdig's point is that an agent can chain recon, credential theft, lateral movement, persistence and destruction 'without the operator possessing deep expertise in any one step'.
